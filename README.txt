# MoveToPlay - Pong with Hand Detection

## Description
This is a Python-based Pong game that uses hand detection for player control. The game leverages Mediapipe for hand tracking and Pygame for rendering the game.

## Prerequisites
- Python 3.9 or higher installed on your machine.
- A webcam for hand detection.

## Installation Instructions
1. Clone or download the project files to your local machine.
2. Open a terminal or command prompt and navigate to the project directory.
3. Install the required dependencies by running:
    pip install -r requirements.txt

## Controls
#- Use your hand to control the paddle.
#- Press `ESC` during the game to return to the menu.
#- The game ends when either the player or the AI scores 5 points.

## Notes
- If the webcam feed does not work, ensure that your webcam is not being used by another application.
- The game is designed to work with a single hand for control.
